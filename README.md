# Password Manager
Приложение написано под android. Предназначено для хранения пары логин-пароль в зашифрованном виде.

Используемый стек технологий
> Kotlin  
> SingleActivity  
> Dagger2  
> RXJava  
> MVVM  
> DataBinding  
> Room
